e-xgen by bluejedi

extjs crud generator for yii

ExGen is inspired and based on giix by mentel, web app generator by derelict and gii-template-collection (gtc), by Herbert Maschke (thyseus).

Dependencies

requirements requirements of using this extension extjs 3.4.0 and Yii 1.1.8 or above

Install

extract to extension folder copy webapp and extjs in extension folder to theme folder copy js folder in extension folder to base path configure config/main php
